## Tipos de selectores
- Selector global `*`
- Selectores por `element`
- Selectores por `id`
- Selectores por `class`

### Selector global

Sirve para seleccionar todos los elementos HTML.


### Selector por `element`

Sirve para seleccionar en base  a un elemento: `h1, table, etc`

### Selector por `class`
